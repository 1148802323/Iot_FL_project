Predictive Maintenance and Federated Learning Baselines on AI4I 2020

This project studies industrial machine failure prediction with a centralized weighted logistic regression baseline and FedAvg baselines across IID, Moderate Non-IID, and Highly Non-IID factory-client settings.

01

Data Preparation

Clean the AI4I 2020 dataset, preserve traceable IDs, and engineer temperature gap, power proxy, and standardized sensor features.

02

Centralized Baseline

Exclude leakage-prone failure-mode columns, train weighted logistic regression, and tune the decision threshold for validation F1.

03

Federated Learning

Treat each factory CSV as a client, train all clients locally, and aggregate model parameters with sample-size weighted FedAvg.

04

Distribution Comparison

Compare IID, Moderate Non-IID, and Highly Non-IID splits to show how client heterogeneity affects model metrics.

Local Reproduction Commands

The complete execution path from data preparation to centralized and federated baselines.

1. Data Preparation

python src/prepare_ai4i_dataset.py --clients 5 --seed 42

2. Centralized Baseline

python src/train_centralized_baseline.py --seed 42 --epochs 500

3. FedAvg baseline

python src/train_fedavg_baseline.py --rounds 50 --local-epochs 5 --seed 42
